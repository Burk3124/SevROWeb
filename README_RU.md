# РСО Севастополь - Веб-сайт Российских Студенческих Отрядов

Веб-сайт для Российских Студенческих Отрядов (РСО) города Севастополя.

## О проекте

Этот веб-сайт создан для информирования студентов о деятельности РСО в Севастополе и предоставления возможности вступления в студенческие отряды.

## Возможности

- Информация о направлениях деятельности РСО
- Информация о процессе вступления в отряды
- Контактная информация
- Адаптивный дизайн для всех устройств

## Технологии

- Laravel 12
- PHP 8.3
- Tailwind CSS (встроенный)
- SQLite (для разработки)

## Установка

### Требования

- PHP >= 8.2
- Composer
- Node.js и NPM (для сборки фронтенда)

### Шаги установки

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Burk3124/SevROWeb.git
cd SevROWeb
```

2. Установите зависимости PHP:
```bash
composer install
```

3. Создайте файл окружения:
```bash
cp .env.example .env
```

4. Сгенерируйте ключ приложения:
```bash
php artisan key:generate
```

5. Создайте базу данных и выполните миграции:
```bash
touch database/database.sqlite
php artisan migrate
```

6. Установите зависимости Node.js и соберите ресурсы:
```bash
npm install
npm run build
```

7. Запустите сервер разработки:
```bash
php artisan serve
```

Сайт будет доступен по адресу: http://localhost:8000

## Разработка

Для разработки с автоматической пересборкой ресурсов:

```bash
npm run dev
```

## Структура проекта

- `resources/views/home.blade.php` - Главная страница сайта
- `routes/web.php` - Маршруты приложения
