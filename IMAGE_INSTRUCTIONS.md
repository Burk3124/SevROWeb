# Инструкции по добавлению изображений

## Фоновое изображение для Hero-секции

### Где разместить:
Сохраните изображение как: `public/images/rso-students-background.jpg`

### Характеристики изображения:
- **Разрешение**: 1920x1080px или выше (рекомендуется 2560x1440px)
- **Формат**: JPG (оптимизированный)
- **Размер файла**: до 500KB (используйте сжатие)
- **Соотношение сторон**: 16:9 или 21:9

### Содержание:
- Группа студентов в форме РСО
- Патриотическая символика
- Достаточно яркое освещение
- Композиция должна оставлять место для текста по центру

### Как активировать:
В файле `resources/views/home.blade.php` (строка ~14) замените:
```html
style="background-image: url('data:image/svg+xml;utf8,<svg viewBox=\"0 0 1200 800\"..."
```
на:
```html
style="background-image: url('/images/rso-students-background.jpg');"
```

## Дополнительные изображения

### Изображение команды в Info-секции:
- Путь: `public/images/rso-team-photo.jpg`
- Размер: 600x480px
- Содержание: фотография студенческого отряда

### Изображения для карусели:
- `public/images/rso-friends.jpg` - Новые друзья (400x300px)
- `public/images/rso-work.jpg` - Опыт работы (400x300px)  
- `public/images/rso-travel.jpg` - Путешествия (400x300px)

### Новостные изображения:
- `public/images/news/` - папка для изображений новостей
- Размер: 300x200px каждое

## Оптимизация изображений

Используйте онлайн-сервисы для сжатия:
- TinyPNG (tinypng.com)
- Squoosh (squoosh.app)
- Или любой другой сервис сжатия изображений

## Примечание

В данный момент используется SVG-заглушка с силуэтами людей. 
После добавления реального изображения сайт будет выглядеть максимально близко к оригинальному макету.
